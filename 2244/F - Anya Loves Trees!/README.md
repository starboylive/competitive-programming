For every vertex, its children are ordered from left to right in increasing order of their indices.</p><p>Anya noticed that if we list the leaves from left to right, their values form a sequence. She wants this sequence to become strictly increasing.</p><p>To achieve this, Anya can perform the following operation: choose any vertex and <span class="tex-font-style-bf">cyclically shift</span> its children to the left$$$^{\text{∗}}$$$. For example, if a vertex has children in the order $$$[1, 2, 3]$$$, after the shift the order becomes $$$[2, 3, 1]$$$. This operation can be applied to any vertices any number of times.</p><p>A visual example of a cyclic shift of the children of vertex $$$1$$$:</p><center> <img class="tex-graphics" src="https://espresso.codeforces.com/3fff36adf711636ed88f8ad73b95cdf075e90197.png" style="max-width: 100.0%;max-height: 100.0%;">  <span class="tex-font-size-small">An example of a cyclic shift of children.</span> </center><p>Help Yura determine whether Anya can make the sequence of integers in the leaves strictly increasing from left to right using such operations.</p><div class="statement-footnote"><p>$$$^{\text{∗}}$$$A cyclic left shift is an operation where all elements are shifted left by $$$1$$$, and the first element becomes the last.</p></div></div><div class="input-specification"><div class="section-title">Input</div><p>The first line contains a single integer $$$t$$$ ($$$1 \le t \le 10^4$$$) — the number of test cases.</p><p>The first line of each test case contains a single integer $$$n$$$ ($$$1 \le n \le 2 \cdot 10^5$$$) — the number of vertices.</p><p>The next line contains $$$n - 1$$$ integers $$$p_2, p_3, \dots, p_n$$$ ($$$1 \le p_i \le n$$$), where $$$p_i$$$ is the parent of vertex $$$i$$$. The children of each vertex are ordered from left to right in increasing order of their indices.</p><p>The third line contains $$$n$$$ integers $$$a_1, a_2, \dots, a_n$$$ ($$$0 \le a_i \le n$$$).
